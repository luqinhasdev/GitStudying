Configuração de Usuário (❁´◡`❁)

> Definição de Nome de Usuário:

$ git config --global user.name "USERNAME"

> Definição de Email de Usuário:

$ config --global user.email "EMAIL"

Inicialização de Repositório (❁´◡`❁)

> Criação de Repositório Git:

$ git init

> Clonagem de Repositório Existente:

$ git clone "URL"

Verificação de Status (❁´◡`❁)

> Verificação do Status Atual:

$ git status

Trabalhando com Arquivos (❁´◡`❁)

> Adcionar Todos os Arquivos no Repositório:

$ git add .

> Adcionar Arquivos Específicos no Repositório:

$ git add "ARQUIVO"

> Remover Arquivos Específicos:

$ git rm "ARQUIVO"

> Desfazer Alterações em Arquivos Específicos:

$ git checkout -- "ARQUIVO"

Trabalhando com "Commits" (❁´◡`❁)

> Realizar uma Commit:

$ git commit -m "MENSAGEM"

> Commitar Todas as Mudanças:

$ git commit -a -m "MENSAGEM"

Trabalhando com "Branches" (❁´◡`❁)

> Criando uma Branch:

$ git branch "NOME"

> Transferindo para uma Branch:

$ git checkout "NOME"

> Criando e Transferindo para uma Branch:

$ git checkout -b "NOME"

> Mostrar Lista de Branches:

$ git branch

> Excluindo Branches Localmente:

$ git branch -d "NOME"

Sincronizando com Repositório Remoto (GitHub) (❁´◡`❁)

> Adcionando o Repositório Remoto:

$ git remote add origin "URL"

> Puxando Alterações do Repositório Remoto:

$ git pull origin "NOME BRANCH"

> Enviando Alterações ao Repositório Remoto:

$ git push origin "NOME BRANCH"

Histórico de Commits (❁´◡`❁)

> Ver o Histórico de Commits:

$ git log

> Ver o Histórico Conciso do Histórico de Commits:

$ git log --oneline

Mesclando Branches (❁´◡`❁)

> Mesclando uma Branch na Branch Atual:

$ git merge "NOME BRANCH"

Rebase de uma Branch (❁´◡`❁)

$ git rebase "NOME BRANCH"

Desfazendo Ações (❁´◡`❁)

> Desfazendo o Último Commit, porém Mantendo Alterações:

$ git reset --soft HEAD~1

> Desfazendo O Último Commit, Mantendo Alterações:

$ git reset --hard HEAD~1

Armazenar Informações Temporariamente (❁´◡`❁)

> Armazenar Alterações não Commitadas:

$ git stash

> Listar Stashes:

$ git stash list

> Aplicar o Stash mais Recente:

$ git stash apply

> Remover o Stash Aplicado:

$ git stash drop

Marcação de Versões

> Criar uma Tag:

$ git tag "NOME TAG"

> Listar Todas as Tags:

$ git tag

> Enviar Tags para o Repositório Remoto:

$ git push origin "NOME TAG"

Trabalhando com Remotes:

> Ver os Repositórios Remotos Configurados:

$ git remote -v

> Remover um Repositório Remoto:

$ git remote remove "NOME REMOTE"

Reverter um Commit (❁´◡`❁)

> Reverter um Commit Específico:

$ git revert "ID COMMIT"
